How the pattern box works — examples
The pattern is a small shape that walks through a pool of notes — the chord tones, or the chord scale (set by Notes from). Position 1 is “home”; 2 3 4… step up the pool; Move by slides the whole shape along after each cycle so the line travels.
1 2 3 4 — run up four pool notes (over Cmaj7 chord tones: C E G B).
1 3 2 4 — a weaving shape · 1 3 5 — an arpeggio.
-1 — a pool note below home · 0 or r — a rest.
<1 / >1 — a chromatic half-step below / above tone 1 (an approach note).
<<1 / >>1 — a chord-scale (diatonic) tone below / above tone 1 — reaches scale notes even while the pool is chord tones.
>>1 <1 1 — an enclosure: scale note above, chromatic below, then the target.
; rotates patterns bar by bar — 1 2 3 4 ; 1 3 5 3 alternates each bar.
chord: / scale: before a pattern sets that bar’s pool — chord: 1 2 3 4 ; scale: 1 2 3 4.
